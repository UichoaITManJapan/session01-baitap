Đây là nội dung trang web

Đây là 1 ví dụ về cách sử dụng thẻ nav và a để tạo menu trong HTML... Trang web này có nột menu đơn giản với năm liên kết đến các trang khác nhau trên trang web. Khi người dùng đăng nhập vào liên kết này, trang web sẽ chuyển hướng về trang web đó.

Đây là trang web của tôi

Chào mừng đến với trang web của tôi.Đây là một ví dụ về cách sử dụng thẻ img để hiển thị hình ảnh trên trang web.

Trên đây là một hình ảnh ngẫu nhiên được tải từ trang web Xổ số miền bắc. Bạn có thể thay đổi kích cỡ hình ảnh, kích thước hình ảnh bằng cách chỉnh độ rộng và chiều cao trong thuộc tính "width" và "height" của thẻ. Bạn có thể thêm một mô tả hình ảnh bằng thuộc tính "alt".
